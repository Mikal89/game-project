# Plant Magic Spell System Explorer

An interactive prototype for exploring a farming-game spell system where every
spell is grown from:

**Role Essence + Material + Environmental Conditions**

The prototype includes:

- A searchable 40-spell spellbook across five progression rungs
- A recipe garden with success, fizzle, dormant, and uncharted outcomes
- Multi-material essence extraction with rarity-scaled yields
- Editable seasonal requirements for Sunlight, Water, Snowfall, Darkness, and Moonlight
- Infinite Water for stream-adjacent plots
- A complete material almanac with spell dependencies

## Run locally

```bash
npm install
npm run dev
```

Then open `http://127.0.0.1:5173/`.
